platform

returns a central class from which all other service stubs are created. It contains the shared settings to be passed to the individual service stubs, for example the root URL of the platform, application credentials, etc.

platform(apiKey: string)
Parameters
apiKey (string) The application ID to identify the client for the ams platform services

behavior

returns a class that encapsulates map behavior functionality. It uses map events and adds functionality such as panning and zooming to the map.

behavior(map: object)
Parameters
map (object) An object representing a previously initialized map events instance
Throws
  • object: InvalidArugmentError if behavior is already instantiated with the specified mapEvents

mapEvents

MapEvents enables the events functionality on the map and on map objects. The class makes it possible to listen to events on map objects such as markers, polylines, polygons, circles and on the map object itself.

mapEvents(map: object)
Parameters
map (object) An object representing the map used for firing events

marker

A Marker is a visual representation of a location on a map in the form of a static bitmap icon.

marker(lat: number, lng: number)
Parameters
lat (number) This property represents the latitude of the point.
lng (number) This property represents the longitude of the point.

icon

A visual representation of the marker

icon(bitmap: (string | HTMLImageElement | HTMLCanvasElement))
Parameters
bitmap ((string | HTMLImageElement | HTMLCanvasElement)) An image URL, an SVG (string), an bitmap image or a canvas.
